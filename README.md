My personal website, built using [Next.js](https://nextjs.org) with the App Router, and [Tailwind CSS](https://tailwindcss.com) for a utility first approach to styling.

The site aims to house information about myself, my CV, and perhaps one day - a blog!

---

## Installation

```
git clone https://github.com/theblindfrog/me && cd me
npm install
```

## Running locally

```
npm run dev
```

## Deployment

Deployed on [Vercel](https://vercel.com). Connect the GitHub repo and Vercel will auto-detect Next.js and deploy automatically.
